# weather_CLI_App
A command-line based weather application built using Python and OpenWeather API.  
It fetches real-time weather data and stores it in a CSV file.

## 🚀 Features

- Get real-time weather by city name
- Displays:
  - Temperature
  - Feels like temperature
  - Weather condition
  - Humidity
  - Wind speed
- Input validation and error handling
- Continuous input support
- Saves data to CSV file for analysis

## 🛠️ Technologies Used

- Python
- Requests library
- OpenWeather API
- CSV module
